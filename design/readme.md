# 思考

## Safari 浏览器

### 功能点：

- 底部常显式导航栏（包含导航、工具按钮和地址输入框）。
  - 包含<前进>、<后退>、<分享>、<书签>、<查看所有tab>
- 打开app显示近期历史页面列表。
- <书签>包含<书签>、<分组>、<全部历史记录>。
- <查看所有tab>可浏览所有tab、切回正常视图、添加tab。
- 可使用滑动手势进行快捷的后退、前进操作。
- 无痕浏览

### 操作特性

- 底部常显式导航栏在上划时就会收起最小化给网页视图让出屏幕，以提高用户体验；同时当下划时该导航栏又会重新展开以等候用户操作。
- 前进、后退仅响应网页路由，当网页退无可退时将不会再后退。
- 只有当用户试图在断网的情况下准备打开网页时才会提示“未连接到互联网”。
- 打开app时判断如果没网则会弹起警告提示、用户可以选择无视或打开设置去打开网络开关。
- 打开app时不管是否有网络都会加载上次未关闭的已缓存页面。



## Chrome 浏览器

### 功能点：

- 底部常显式导航栏（包含导航、工具按钮和地址输入框）。
  - 包含<前进>、<后退>、<添加tab>、<查看所有tab>、<其他功能>
- 打开app显示推荐网页、快捷操作列表。
- <其他功能>包含<书签>、<历史记录>、<新变化提示>、<设置>。
- 可使用滑动手势进行快捷的后退、前进操作。

### 操作特性

- 底部常显式导航栏在上划时就会收起完全隐藏给网页视图让出屏幕，以提高用户体验；同时当下划时该导航栏又会重新展开以等候用户操作。
- 顶部地址栏会缩起最小化以达成上述目的。
- 前进、后退响应网页路由，当网页退无可退时会后退至首页。
- 只有当用户试图在断网的情况下准备打开网页时才会提示“未连接到互联网”。
- 打开app时判断如果没网则会弹起警告提示、用户可以选择无视或打开设置去打开网络开关。
- 打开app时不管是否有网络都会加载上次未关闭的已缓存页面。



# 设计

- 打开app时显示《首页》。
  - 展示地址输入、一些快捷操作、推荐地址等内容，等待用户后续操作。
  - 检测当前网络连接，如果断网则警告用户状态并可引导前往系统设置页。
  - 可使用扫码快捷输入。
  - 输入地址后前往《主显示页》
- 《主显示页》用于展示网页主体。
- 顶部常显式导航栏。
  - 包含<后退、前进>、<刷新>、<重新输入网址>、<其他按钮>。
  - 加载网页完成后，滑动屏幕或3s后自动收起。收起状态时除非点击展开否则不会展开。同时提示首次运行的用户上述事项。
  - <后退>仅响应网页、当页面退无可退时不再后退。
  - 可使用滑动手势替代后退、前进操作。
  - 点进刷新后重新加载当前地址。
  - 点击<重新输入网址>回到《首页》。
  - 后续功能都放在<其他按钮当>当中。
- 不支持多tab挂载。
- 可自动存储浏览历史，有效期为十年。
- 可对某些网址进行收藏，有效期永久。
